## Getting Started

```bash
git clone https://github.com/HimitsuTH/TIC-TAC-TOE.git

#cd to folder frontend
cd frontend
#install node module
npm install 
#run this project
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
